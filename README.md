# Bloom_Game
🌸 Bloom Garden — idle farming RPG with weather, mutations, lootboxes. Grow 30+ crops. Weather gives bonuses &amp; mutations. Sell manually or auto. Shop: seeds, tools, crates, upgrades. Crates: free, paid, XP. Wheel: 3 free spins/2h. Garden Pass: 35 levels. Profile &amp; leaderboards. Tap to plant, harvest, sell. Single HTML, mobile-friendly. MIT.
